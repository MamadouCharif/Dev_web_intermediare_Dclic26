Dans le cadre de cette activité, j’ai vérifié les performances de la page à l’aide de Chrome DevTools (onglet Network).

Deux optimisations ont été réalisées :

1. Regroupement des styles CSS communs afin de réduire les répétitions et améliorer la lisibilité du code.

2. Suppression d’une variable CSS inutilisée afin d’alléger le fichier CSS.

Résultats observés :

* Nombre de requêtes : 1
* Taille des ressources avant optimisation : 3,8 ko
* Données transférées avant optimisation : 3,8 ko
* Taille des ressources après optimisation : 3,1 ko
* Données transférées après optimisation : 242 octets

Ces améliorations ont permis de réduire la taille des ressources et d’obtenir un code plus propre tout en conservant le même affichage et les mêmes fonctionnalités.
